# Getting and Cleaning Data - Data Science Course Project

This is the course project for the Getting and Cleaning Data Coursera course. Main R script, run_analysis.R, contains the following:

Checks for data existence and do a download and unpicking if needed.
Loads the activity and feature info, alongside with all training and test datasets
Loads the activity and subject data for each dataset, and merges those columns with the dataset
Merges the two datasets
Creates a tidy dataset that consists of the average (mean) value of each variable for each subject and activity pair.
The end result is shown in the file tidySet.txt.
